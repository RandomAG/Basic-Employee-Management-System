# Basic-Employee-Management-System
A very basic employee management system made using Java and Springboot.

Steps to run it:

1. To run this project - 
   First kill any process in port 8081:
   `npx kill-port 8081`
2. Next, run the project in the root folder (example: D:/Projects/Basic-Employee-Management-System):
   `.\mvnw spring-boot:run`
3. Then go to the link given below:
   `http://localhost:8081/login.html`
4. You can sign in with these credentials:
   Admin - `admin` / `admin123`
   User - `user` / `user123`
